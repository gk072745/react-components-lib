# Range

A flexible and customizable dual-thumb range slider component that allows users to select a range of values between a minimum and maximum.

## Overview

The BasicRange component provides:

- Dual thumb control for selecting min and max values
- Five size variants (xs, sm, md, lg, xl)
- Six color variants (default, primary, success, warning, danger, info)
- Keyboard navigation support
- Touch support for mobile devices
- Configurable step increments
- Optional thumb labels
- Disabled and readonly states
- Accessible with ARIA attributes
