# Textarea

A comprehensive textarea component with auto-grow, validation, character counter, and extensive customization options.

## Overview

The BasicTextarea component provides:

- Auto-grow functionality that adjusts height based on content
- Character counter with optional maximum length
- Built-in validation system with custom rules
- Custom icons support (prepend, prependInner, append, appendInner)
- Multiple states (disabled, readonly, loading)
- No resize option
- Full keyboard navigation and accessibility support
