# File Input

A comprehensive file input component with support for single and multiple file selection, drag-and-drop, file validation, chip display, and custom icons.

## Overview

The BasicFileInput component provides:

- Single and multiple file selection
- Drag-and-drop file upload support
- File validation (max files, max size, file types)
- Chip-based file display for multiple files
- File size calculation and display
- Custom prepend/append icons
- Loading and disabled states
- Readonly mode
- Paste support for files
- Accessible design with proper form semantics
