# Radio

A versatile and customizable radio button component that supports both single and multiple selection modes with flexible styling options.

## Overview

The BasicRadio component provides:

- Single and multiple selection modes
- Five size variants (xs, sm, md, lg, xl)
- Six color variants (default, primary, success, warning, danger, info)
- Toggle mode for deselection in single selection
- Custom icon support via render functions
- Disabled and readonly states
- Flexible value types (string, number, boolean)
- Custom value comparison function
