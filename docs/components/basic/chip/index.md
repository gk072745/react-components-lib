# Chip

A versatile chip component for displaying compact information, tags, and interactive elements with support for multiple variants, custom content slots, and closable functionality.

## Overview

The BasicChip component provides:

- Compact display of information, tags, or labels
- Multiple visual variants (default, primary, success, warning, danger, info)
- Three variant types (solid, outlined, filled)
- Support for closable/removable chips with custom delete handling
- Custom content with prepend, append, and children slots
- Flexible data handling (strings or objects with customizable keys)
- Interactive states with click and delete handlers
- Disabled state support
- Accessible design with proper ARIA attributes
