# Chip

A versatile chip component for displaying compact information, tags, and interactive elements.

## Overview

The Chip component provides:

- Compact display of information, tags, or labels
- Multiple visual variants and styling options
- Support for closable/removable chips
- Custom content with prepend and append slots
- Flexible data handling (strings or objects)
- Interactive states with click and delete handlers

## When to Use

Use the Chip component when you need to:

- Display tags, categories, or labels
- Show selected items in multi-select interfaces
- Create filter chips for search and filtering
- Display user selections or preferences
- Show status indicators or badges
- Create interactive elements with custom content

## Key Features

- **Flexible Data**: Support for both string and object data with customizable text/value keys
- **Multiple Variants**: Default, primary, success, warning, danger, and info variants
- **Variant Types**: Solid, outlined, and filled styling options
- **Closable Chips**: Optional close button with custom delete handling
- **Custom Content**: Prepend, append, and children slots for custom rendering
- **Interactive States**: Click handlers and disabled states
- **Accessibility**: Proper ARIA attributes and keyboard navigation
- **Responsive Design**: Adapts to different screen sizes
