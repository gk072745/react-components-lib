# Menu

A flexible menu component with positioning, trigger types, and portal rendering for dropdown menus, context menus, and popover content.

## Overview

The BasicMenu component provides:

- Multiple placement options (top, bottom, left, right, and variants)
- Click and hover trigger types
- Automatic collision detection and placement flipping
- Portal rendering to document body
- Viewport boundary detection and adjustment
- Customizable width and trigger width matching
- Keyboard support (ESC to close)
- Outside click detection
- Smooth open/close animations
- Accessible design with proper event handling
