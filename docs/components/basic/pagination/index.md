# Pagination

A comprehensive pagination component with smart ellipsis, flexible navigation, and modern React 19 support.

## Overview

The Pagination component provides:

- Smart ellipsis handling for large page ranges
- Flexible navigation controls (first/last, prev/next)
- Multiple size variants and color themes
- Comprehensive accessibility support
- Event handling for different actions
- Responsive design for all screen sizes

## When to Use

Use the Pagination component when you need to:

- Navigate through large datasets
- Display paginated content
- Provide user-friendly page navigation
- Handle data tables with pagination
- Create search result pagination
- Implement content pagination

## Key Features

- **Smart Ellipsis**: Automatically shows ellipsis for large page ranges
- **Flexible Navigation**: Optional first/last and previous/next buttons
- **Size Variants**: Small, default, and large sizes
- **Color Themes**: Primary and secondary variants
- **Rounded Style**: Optional rounded button appearance
- **Disabled State**: Complete disabled functionality
- **Accessibility**: Proper ARIA labels and keyboard support
- **Event Handling**: Separate callbacks for different actions
- **Customizable**: Configurable total visible pages
- **Responsive**: Adapts to different screen sizes
