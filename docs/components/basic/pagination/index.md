# Pagination

A customizable pagination component with smart ellipsis handling, flexible navigation controls, and comprehensive accessibility support.

## Overview

The BasicPagination component provides:

- Smart ellipsis handling for large page ranges
- Flexible navigation controls (first/last, prev/next)
- Multiple size variants (small, default, large)
- Color themes (primary, secondary)
- Rounded button style option
- Disabled state support
- Comprehensive event handling
- Accessible design with proper ARIA attributes
