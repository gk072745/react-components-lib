# Side Panel

A flexible side panel component with navigation support, overlay interaction, breadcrumb integration, and popup capabilities.

## Overview

The SidePanel component provides:

- Slide-in navigation panel from the left
- Overlay backdrop with click-to-close functionality
- Navigation items with icons and active state highlighting
- Breadcrumb navigation integration
- Popup integration for additional content
- Smooth animations and transitions
- Customizable styling and className support
- Full viewport height coverage
- Keyboard navigation support
