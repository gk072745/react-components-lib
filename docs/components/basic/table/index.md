# BasicTable

A powerful and flexible table component for displaying structured data with advanced features like sorting, pagination, and interactive functionality.

## Overview

The BasicTable component provides:

- Sortable columns with visual indicators
- Built-in pagination support
- Interactive cell click handlers
- Copy functionality for cell values
- Custom cell rendering via `domFunc`
- Hover effects and alternate row/column colors
- Flexible column widths and alignment
- Keyboard navigation and accessibility support
- Responsive design
