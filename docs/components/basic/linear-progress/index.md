# Linear Progress

A flexible linear progress bar component supporting determinate and indeterminate modes with customizable variants, height, rounding, and positioning.

## Overview

The LinearProgress component provides:

- Determinate mode with precise progress values
- Indeterminate mode with animated progress for unknown durations
- Multiple color variants (default, primary, success, warning, danger, info)
- Customizable height and styling
- Rounded corners option
- Absolute positioning for top-aligned progress bars
- Smooth transitions and animations
- Accessibility features with proper focus styles
