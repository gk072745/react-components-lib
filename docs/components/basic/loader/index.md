# Loader

A flexible loader overlay component with SVG spinner or custom image support, supporting both local and full-screen positioning with customizable size, stroke width, and color variants.

## Overview

The Loader component provides:

- SVG spinner with smooth animations
- Custom image loader support
- Local (relative) or full-screen (fixed) positioning
- Multiple color variants (default, primary, success, warning, danger, info)
- Customizable size and stroke width
- Custom children support for complete override
- Accessibility features with reduced motion support
- Smooth animations and transitions
