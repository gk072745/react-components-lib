# Toast

A versatile toast notification system for displaying temporary messages, alerts, and feedback to users.

## Overview

The Toast component provides:

- Multiple notification types (success, error, warning, info)
- Flexible positioning (top-left, top, top-right, bottom-left, bottom, bottom-right)
- Customizable timeouts and persistent notifications
- Global state management with independent toast instances
- Smooth animations and transitions
- Responsive design with rem-based positioning
- Accessibility features with proper ARIA attributes

## When to Use

Use the Toast component when you need to:

- Show success messages after user actions
- Display error notifications for failed operations
- Provide warning messages for important information
- Show informational updates or tips
- Give feedback on form submissions
- Display system notifications
- Show temporary status updates

## Key Features

- **Multiple Types**: Success, error, warning, and info notification types
- **Flexible Positioning**: Six different positions with customizable offsets
- **Global Management**: Centralized toast state with independent instances
- **Customizable Timeouts**: Per-toast or global timeout settings
- **Persistent Notifications**: Optional persistent toasts that don't auto-dismiss
- **Smooth Animations**: CSS transitions for enter/exit animations
- **Responsive Design**: Rem-based positioning that adapts to different screen sizes
- **Accessibility**: Proper ARIA attributes and keyboard navigation
- **Custom Content**: Support for primary and secondary messages
- **Icon Support**: Optional custom icons for each toast type
