## Overview

The BasicToast component provides:

- Multiple notification types (success, error, warning, info)
- Flexible positioning (top-left, top, top-right, bottom-left, bottom, bottom-right)
- Customizable timeouts and persistent notifications
- Global state management with independent toast instances using React 18's `useSyncExternalStore`
- Maximum notifications per position (defaults to 5, configurable)
- Smooth animations and transitions
- Responsive design with rem-based positioning
- Accessibility features with proper ARIA attributesnd transitions
- Responsive design with rem-based positioning
- Accessibility features with proper ARIA attributes
