# Snackbar

A flexible and customizable snackbar notification component that displays temporary messages to users.

## Overview

The BasicSnackbar component provides:

- Multiple notification types (success, error, warning, info)
- Flexible positioning (6 positions: top/bottom + left/center/right)
- Secondary messages for additional context
- Auto-dismiss with configurable timeout
- Persistent mode for important notifications
- Custom icons support
- Global state management via useSnackbar hook
- Smooth animations
- Accessible design with ARIA attributes
