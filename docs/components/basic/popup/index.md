# Popup

A customizable popup component for displaying modal content with outside click handling and flexible sizing.

## Overview

The BasicPopup component provides:

- Customizable dimensions with height and width props
- Outside click detection and handling
- Simple and clean modal overlay with backdrop blur
- Flexible content rendering
- Fixed positioning with centered layout
- Event propagation control
