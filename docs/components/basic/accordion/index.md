# Accordion

A customizable accordion component with support for expandable/collapsible content sections.

## Overview

The Accordion component provides:

- Expandable and collapsible content sections
- Customizable header with title and icons
- Smooth animations and transitions
- Accessible design with proper ARIA attributes
- Support for custom prepend and append content

## When to Use

Use the Accordion component when you need to:

- Organize content into collapsible sections
- Create FAQ pages or help documentation
- Show/hide detailed information on demand
- Reduce visual clutter by hiding content until needed
- Create navigation menus with expandable sub-items

## Key Features

- **Expandable Content**: Show/hide content sections with smooth animations
- **Custom Headers**: Customize the header with title and optional icons
- **Prepend/Append Slots**: Add custom content before or after the title
- **Initial State**: Control whether accordion starts open or closed
- **Disabled State**: Disable accordion interaction when needed
- **Keyboard Navigation**: Full keyboard support with Enter/Space keys
- **Accessibility**: Proper ARIA attributes and screen reader support
- **Event Callbacks**: Handle state changes with onToggle callback
